## breadboard

breadboard는 공용 Bulletin Board System으로서 개발자와 사용자를 대상으로 개발하고있습니다.

BBS모듈을 Restful API와 View Template 제공합니다. 


## 개발 및 구축 환경

- SQL Server 2016 (2008+)
- JDK 8+
- spring boot 2.0
- Tomcat 9+
